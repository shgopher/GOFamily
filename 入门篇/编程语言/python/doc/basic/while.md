# 循环

## 基本语法
- for in
  - rang()
- while
  - break
  - continue

## for in

举例子
```python
for x in [1,2,3]:
    print(x)
```
## while
举例子
```python

while x < 0:
    print(x)
    x --
```
## break 和 continue

```python
for x in [1,2,3,4,5,6,7]:
    if i > 4:
```
